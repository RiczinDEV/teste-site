PRIME VIDEO - PROTÓTIPO PREMIUM FULL (30 filmes) - XAMPP

Gerado em: 2025-11-07T02:40:10.911185 UTC

O que contém:
- Site PHP + MySQL pronto para XAMPP com 30 filmes de exemplo
- Login / cadastro (PHP), catálogo dividido por categorias, hero/carousel, player local
- Imagens de exemplo (thumbnails e hero) e arquivos de vídeo placeholder em /videos
- Script / includes que criam o banco automaticamente na primeira execução (includes/db.php)
- Arquivo SQL dump opcional: prime_prototipo_dump.sql

Como rodar no XAMPP (Windows):
1) Copie a pasta 'prime_prototipo_premium_full30_xampp' para C:\xampp\htdocs\
2) Abra o XAMPP Control Panel e inicie Apache e MySQL.
3) Abra no navegador: http://localhost/prime_prototipo_premium_full30_xampp/
4) Registre-se (register.php) e faça login. Navegue pelo catálogo e clique em "Assistir".
5) Substitua os arquivos em /videos pelos seus .mp4 reais (nomes: filme1.mp4 ... filme30.mp4).

Observações de segurança e performance:
- Este é um protótipo acadêmico. Para produção, implemente validações, proteção a uploads, HTTPS e otimizações de streaming.
